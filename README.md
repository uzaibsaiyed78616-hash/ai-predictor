**Smart AI Prediction System**

**A full-stack AI-powered prediction system using Machine Learning, Flask, Node.js, and React.**

---

## Features

- Salary Prediction
- House Price Prediction
- Student Performance Prediction
- Basic Stock Trend Prediction

---

## Tech Stack

### Frontend
- React (Vite)
- Axios

### Backend
- Node.js (Express API Gateway)
- Flask (ML API Server)

### Machine Learning
- Python
- Scikit-learn
- Pandas

---

## Project Structure


ai-predictor/
│
├── backend/ (Node.js API)
├── ml-model/ (Flask + ML models)
├── frontend/ (React UI)
└── README.md


---

## How to Run Project

### 1. Backend (Node.js)

cd backend
node server.js


---

### 2️. ML Server (Flask)

cd ml-model
python app.py


---

### 3️. Frontend (React)

cd frontend
npm install
npm run dev


---

## Models Used

- Linear Regression (Salary Prediction)
- Simple Regression (House Price)
- Student Performance Model
