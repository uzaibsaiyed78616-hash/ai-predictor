# Smart AI Prediction System

## Features
- Salary Prediction
- House Price Prediction
- Student Performance Prediction

## Tech Stack
- Python (Flask)
- Node.js (Express)
- React
- Scikit-learn

## How to Run

### 1. Train Models
python train.py

### 2. Run Flask API
python app.py

### 3. Run Backend
node server.js

### 4. Run Frontend
npm start
